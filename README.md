# Nike Sales Data Analysis

##  Description
Ce projet consiste à analyser un jeu de données de ventes (produits Nike) en appliquant un **pipeline complet de Data Analysis** :

1. Exploration des données avant traitement  
2. Nettoyage et prétraitement des données  
3. Visualisation des données après traitement  

L'objectif est de comprendre la qualité des données, corriger les erreurs, gérer les valeurs manquantes et analyser la rentabilité des ventes.
